# 📺 youtube-comments-overlay
YouTube のコメントを **OBS に表示できるオーバーレイツール** です。

---

## 📥 ダウンロード & インストール

以下のページから最新版をダウンロードできます：

👉 **https://github.com/suzunayui/youtube-comments-overlay/releases/**

ダウンロードするファイル名の例：

YouTubeOverlay-2025.12.7.exe


インストールしてアプリを起動すると、  
**複数の HTML ファイルが入ったフォルダ** が自動で開きます。

---

## 🎬 OBS への追加方法

1. 開いたフォルダの  
overlay.html  
niconico.html  
supers.html  
concurrent.html  
effects.html  
これらの HTML を **OBS にまとめてドラッグ＆ドロップ**します。

2. アプリ側の設定画面にある  
**「チャンネルID / @ハンドル / 動画ID」**  
のどれか1つを入力して、

3. **「チャット取得開始」** ボタンを押すと  
OBS 上にライブチャットが表示されます！

---

## ✨ 使えるオーバーレイ一覧

| ファイル名          | 内容 |
|--------------------|------|
| **overlay.html**    | 通常のコメントオーバーレイ |
| **niconico.html**   | ニコニコ動画風の弾幕表示 |
| **supers.html**     | スパチャ・メンバーのみ表示 |
| **concurrent.html** | 同時接続・高評価バッジ表示 |
| **effects.html**    | 花火・雪エフェクト（キーワードで発火） |

---

## 💡 補足
- OBS にはドラッグ＆ドロップするだけで追加できます  
- HTML の大きさは OBS 側で自由に調整できます  
- アプリを閉じてもオーバーレイはそのまま使えます  
- effects.html はチャット内キーワードで発火します  
  - 花火: 花火/はなび/hanabi/firework(s)  
  - 雪: 雪/ゆき/snow
