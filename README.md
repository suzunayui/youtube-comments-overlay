# 📺 youtube-comments-overlay
YouTube のコメントを **OBS に表示できるオーバーレイツール** です。

---

## 📥 ダウンロード & インストール

以下のページから最新版をダウンロードできます：

👉 **https://github.com/suzunayui/youtube-comments-overlay/releases/**

ダウンロードするファイル名の例：

YouTubeOverlay-2025.12.5.exe


インストールしてアプリを起動すると、  
**3つの HTML ファイルが入ったフォルダ** が自動で開きます。

---

## 🎬 OBS への追加方法

1. 開いたフォルダの  
index.html
niconico.html
supers.html
この3つの HTML を **OBS にまとめてドラッグ＆ドロップ**します。

2. アプリ側の設定画面にある  
**「チャンネルID / @ハンドル / 動画ID」**  
のどれか1つを入力して、

3. **「チャット取得開始」** ボタンを押すと  
OBS 上にライブチャットが表示されます！

---

## ✨ 使えるオーバーレイ一覧

| ファイル名        | 内容 |
|------------------|------|
| **index.html**    | 通常のコメントオーバーレイ |
| **niconico.html** | ニコニコ動画風の弾幕表示 |
| **supers.html**   | スパチャ・メンバーのみ表示 |

---

## 💡 補足
- OBS にはドラッグ＆ドロップするだけで追加できます  
- HTML の大きさは OBS 側で自由に調整できます  
- アプリを閉じてもオーバーレイはそのまま使えます
